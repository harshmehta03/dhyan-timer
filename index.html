Microphone Access
For Omkar counting to work, the app needs microphone access. Follow these steps if Chrome isn't asking for permission.
1
Open in Chrome (not WhatsApp browser)
In WhatsApp, tap ⋮ → Open in Chrome
2
When Chrome asks for mic permission, tap Allow
3
If you already denied: tap the 🔒 lock icon in Chrome's address bar → Permissions → Microphone → Allow
4
If still blocked: Chrome Settings → Site Settings → Microphone → find the file → Allow
